img-ascii

概要 #

img-asciiは、画像を文字だけで表現された「ASCIIアート」に変換するソフトウェアです。

ASCIIアートとは #

ASCIIアートとは、文字や記号を組み合わせて画像を表現する技術です。コンピュータが普及する以前から利用されており、現在でも装飾やアート作品として親しまれています。

このソフトウェアの特徴 #

  • Rustで開発された高速なCLIツール
  • 画像をグレースケール化して文字へ変換
  • コマンド一つでASCIIアートを生成可能

利用方法 #

事前にRustをインストールしたうえで、次のコマンドを実行します。

cargo run -- 画像ファイルへのパス

開発背景 #

本プロジェクトはRustの学習と画像処理技術の理解を目的として開発しました。
