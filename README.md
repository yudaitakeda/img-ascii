ターミナルで指定画像をascii画像で描画

## Description

A simple CLI tool to convert images into ASCII art, powered by Rust.
ターミナル上で画像をグレースケール変換し、輝度に基づいた文字割り当てを行うことでASCIIアートを生成します。
軽量で依存関係の少ない実装を目指したツールです。
